## 💡 Tips for Data Collection

1. **Start with Projects:** Your projects are the most important. Gather all project details first.

2. **Use Existing Materials:**
   - Check your GitHub repositories for README files
   - Review your LinkedIn profile
   - Look at project documentation you've written

3. **Take Screenshots:**
   - If projects are live, take fresh screenshots
   - Use browser dev tools to capture at different screen sizes
   - Ensure screenshots are high quality

4. **Write Content First:**
   - Don't wait for design to write content
   - Use the CONTENT_TEMPLATES.md file
   - Get feedback on written content before building

5. **Organize as You Go:**
   - Create the folder structure early
   - Name files consistently
   - Keep a master document with all links
